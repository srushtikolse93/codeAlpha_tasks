# Student Grade Tracker

## Description
A Java application that helps manage student grades. The program allows users to enter student names and grades, then calculates the average score, highest score, and lowest score.

## Features
- Add student names and grades
- Store data using ArrayList
- Calculate average score
- Find highest score
- Find lowest score
- Display a student summary report

## Technologies Used
- Java
- ArrayList
- Object-Oriented Programming (OOP)
- Scanner Class
