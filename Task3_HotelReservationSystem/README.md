# Hotel Reservation System

## Description

A Java-based Hotel Reservation System that allows users to view rooms, book rooms, cancel reservations, simulate payments, and view booking details.

## Features

* View available rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
