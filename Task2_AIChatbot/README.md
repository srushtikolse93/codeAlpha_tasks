# AI Chatbot

## Description

A simple AI Chatbot built using Java that interacts with users through predefined responses.

## Features

* Interactive conversation
* Responds to common questions
* Case-insensitive input handling
* Exit option using "bye"

## Technologies Used

* Java
* Scanner Class
* Conditional Statements
* Loops

## How to Run

Compile:
javac AIChatbot.java

Run:
java AIChatbot

## Sample Output

=================================

Welcome to AI Chatbot

Type 'bye' to exit.

You: hello
